Minimax Rule

If the a priori probabilities are unknown, we treat them as variables and consider the worst-case set of a priori probabilities for each possible rule, then select the rule with the lowest worst-case probability of error.

Consider a BSC with 0s and 1s - let the a priori probability of 0 be denoted P(0).
There are four possible decision rules - plot the probability of error for each one as a function of P(0). You'll find it's a set of lines in the box [0,0]-[1,1]
The minimax rule is the one with the lowest maximum.
